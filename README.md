# ☕ Starbucks Landing Page — Front-End Project

Este projeto é uma **landing page responsiva inspirada na identidade visual da Starbucks**, desenvolvida com foco em **layout moderno, organização de código, boas práticas de CSS e experiência do usuário (UX)**.

O objetivo é demonstrar domínio de **HTML5, CSS3 e design responsivo**, entregando uma interface elegante, funcional e adaptável a diferentes dispositivos.

---

## 🚀 Demonstração do Projeto

Landing page institucional contendo:
- Header com busca e menu
- Banner promocional (Happy Hour)
- Seções de destaque com grid layout
- Área institucional
- Footer responsivo

> Projeto totalmente **mobile-first**, adaptado para smartphones, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível  
- **CSS3** — Estilização avançada e responsiva  
- **Google Fonts** — Montserrat & Oswald  
- **CSS Grid & Flexbox** — Layout moderno e organizado  
- **Media Queries** — Responsividade em múltiplos breakpoints  

---

## 🎯 Principais Pontos Fortes (Diferenciais)

### ✅ Layout Profissional e Responsivo
- Adaptado para **768px, 480px e 360px**
- Excelente leitura e usabilidade em dispositivos móveis
- Botões e áreas de toque respeitando padrões mobile (mín. 44px)

### ✅ Organização de Código
- CSS bem estruturado por seções (`header`, `banner`, `area1`, `area2`, `area3`, `footer`)
- Nomenclatura clara e padronizada
- Separação lógica entre estrutura (HTML) e estilo (CSS)

### ✅ Boas Práticas de CSS
- Reset global (`box-sizing`, `margin`, `padding`)
- Uso inteligente de cores, fontes e espaçamentos
- Reaproveitamento de classes (`.button`, `.bar`, `.container`)

### ✅ Design Moderno
- Tipografia bem definida
- Uso equilibrado de imagens e espaços em branco
- Grid layouts que valorizam o conteúdo visual
- Hierarquia visual clara (títulos, textos e CTAs)

### ✅ Experiência do Usuário (UX)
- Header limpo e funcional
- Campo de busca intuitivo
- Call to Actions claros (“Learn More”)
- Conteúdo escaneável e agradável

---

## 📱 Responsividade

O projeto foi pensado para funcionar perfeitamente em:

- 📱 Smartphones  
- 📱 Tablets  
- 💻 Desktops  
- 🖥️ Telas grandes  

Cada breakpoint foi cuidadosamente ajustado para manter:
- Legibilidade
- Alinhamento
- Proporção de imagens
- Facilidade de navegação
